# HTML & CSS MCQ Questions

---

## 1. Which HTML tag is used to create a hyperlink?
A) `<link>`  
B) `<a>`  
C) `<href>`  
D) `<url>`  
Answer: `<a>`

---

## 2. Which CSS property is used to make text bold?
A) font-style  
B) font-weight  
C) text-bold  
D) font-size  
Answer: `font-weight`

---

## 3. Which HTML5 element is used for navigation links?
A) `<section>`  
B) `<nav>`  
C) `<aside>`  
D) `<header>`  
Answer: `<nav>`

---

## 4. Which CSS property creates rounded corners?
A) border-radius  
B) corner-radius  
C) radius  
D) border-style  
Answer: `border-radius`

---

## 5. Which display property hides an element but keeps its space?
A) display:none  
B) visibility:hidden  
C) opacity:0  
D) hidden  
Answer: `visibility:hidden`

---

## 6. Which CSS unit is relative to the root font size?
A) px  
B) %  
C) rem  
D) vh  
Answer: `rem`

---

## 7. Which HTML input type is used for email validation?
A) text  
B) mail  
C) email  
D) validation  
Answer: `email`

---

## 8. Which CSS property controls the stacking order of elements?
A) position  
B) z-index  
C) overflow  
D) opacity  
Answer: `z-index`

---

## 9. Which selector targets only direct child elements?
A) +  
B) ~  
C) >  
D) *  
Answer: `>`

---

## 10. Which HTML element is semantically correct for the main content?
A) `<content>`  
B) `<body>`  
C) `<main>`  
D) `<section>`  
Answer: `<main>`

---

## 11. Which CSS layout module provides two-dimensional layouts?
A) Flexbox  
B) Grid  
C) Float  
D) Inline-block  
Answer: `Grid`

---

## 12. Which property is used to create smooth animations?
A) animation  
B) transition  
C) transform  
D) motion  
Answer: `transition`

---

## 13. Which HTML attribute improves accessibility for images?
A) src  
B) alt  
C) title  
D) caption  
Answer: `alt`

---

## 14. Which CSS pseudo-class selects the first child element?
A) :first  
B) :first-child  
C) :child(1)  
D) :nth(0)  
Answer: `:first-child`

---

## 15. Which meta tag makes websites responsive?
A) `<meta charset="UTF-8">`  
B) `<meta name="viewport" content="width=device-width, initial-scale=1.0">`  
C) `<meta responsive="true">`  
D) `<meta device="mobile">`  
Answer: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
